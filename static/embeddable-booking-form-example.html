Integration Guide

How to Use the Embeddable Form

Follow these steps to add the booking form to your website:

  1. Add the script tag to your HTML page
  2. Create a container element where you want the form to appear
  3. Initialize the form with your business ID
<!-- 1. Add the script tag -->
<script src="https://cleaningbizai.com/static/js/embeddable-booking-form.js"></script>

<!-- 2. Create a container element -->
<div id="booking-form-container"></div>

<!-- 3. Initialize the form -->
<script>
    document.addEventListener('DOMContentLoaded', function() {
        new CleaningBizBookingForm({
            businessId: 'your-business-id-here',
            targetElement: '#booking-form-container',
            apiBaseUrl: 'https://cleaningbizai.com',
            onSuccess: function(result) {
                console.log('Booking created successfully:', result);
            },
            onError: function(error) {
                console.error('Error creating booking:', error);
            }
        });
    });
</script>

Configuration Options

The CleaningBizBookingForm constructor accepts the following options:

Option Type Description Required
businessId String The ID of the business to create bookings for Yes
targetElement String CSS selector for the container element Yes
apiBaseUrl String Base URL for the API No (default: https://cleaningbizai.com)
onSuccess Function Callback function when booking is successful No
onError Function Callback function when booking fails No

Overview

The CleaningBiz Booking Form is a JavaScript library that allows you to embed a booking form on your website. The form connects to the CleaningBiz API to retrieve business information, calculate prices, and create bookings.

Requirements

  • A modern web browser (Chrome, Firefox, Safari, Edge)
  • A valid business ID from the CleaningBiz platform
  • Internet connection to access the API

Customization

You can customize the appearance of the form by adding your own CSS styles. The form uses the following CSS classes:

  • cleaningbiz-booking-form - The main container for the form
  • cleaningbiz-form-section - Each section of the form
  • cleaningbiz-form-row - A row of form fields
  • cleaningbiz-form-group - A form field group
  • cleaningbiz-addon-item - An add-on item
  • cleaningbiz-price-summary - The price summary section
  • cleaningbiz-submit-btn - The submit button

Events

The booking form emits the following events:

  • booking:success - Fired when a booking is successfully created
  • booking:error - Fired when there is an error creating a booking
  • form:loaded - Fired when the form is loaded and ready
  • price:updated - Fired when the price is updated

You can listen for these events using the following code:

document.addEventListener('booking:success', function(event) {
  console.log('Booking created:', event.detail);
});

Troubleshooting

Common issues and solutions:

Form doesn't load

  • Check that the business ID is correct
  • Verify that the API base URL is accessible
  • Check the browser console for errors

Booking creation fails

  • Ensure all required fields are filled out
  • Check that the API base URL is correct
  • Verify that the business is active and approved
